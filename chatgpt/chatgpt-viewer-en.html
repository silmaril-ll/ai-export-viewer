Welcome 🎉 ChatGPT Export Viewer 🎉。

Load the conversations.json file exported from chat.openai.com to comfortably browse your chat history locally.

Cool Features Edition (v4.5) highlights:

  • 🗂️ Navigation tabs：Switch between panels quickly, and edit custom names anytime
  • 🔍 Enhanced search：Search full text or titles, and jump quickly with ←/→ (Shift + Enter / Enter)
  • ⭐ Favorites：Favorite entire conversations or individual messages; click to jump back
  • ☁️ Word frequency analysis：Analyze Chinese/English/mixed frequencies for current conversation or all records; generate a word cloud + frequency table (click a word to search)
  • 🕓 Message timestamp toggle：Off by default; enable it in the "Tools" tab
  • 🧭 UI improvements：Collapsible sidebar (S key), draggable width, and linked light/dark + bubble palettes
  • ⚡ Performance optimizations：Auto-load last cache and use local IndexedDB caching for faster large-file loading

About data security

This tool runs entirely locally in your browser. No network, no uploads. Your chats belong only to you.
